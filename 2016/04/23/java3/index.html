JAVA进阶——插入排序

  • java

代码

分析

刚开始接触插入排序时，有点不太理解，不理解的是谁和谁进行比较，插入到哪？松哥给我讲了两遍，可惜还是似懂非懂。自己又思考了下终于弄懂。（自己太笨了，呵呵）
插入排序，简单来说就是将数字插入其应该在的位置。插入时，先选取待插入的数字（从数组下标i=1开始），将待插入的数字与其前面（下标小于i的数字）进行比较，当前面的数字大于待选数字时则将前面的数字后移。
