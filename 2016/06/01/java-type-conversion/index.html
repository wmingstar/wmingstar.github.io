JAVA进阶——类型转换

  • java

在学习java过程中，有些知识很简单，但是却很重要。这些简单的知识是学好javaa的基石，需要认真对待。在做刷题过程中经常遇到一些需要进行类型转换的问题，所以对类型转换相关的知识进行整理如下。

String类与字符型相互转换

String转char:

用String的toCharArray()方法可以将字符串转换为字符数组。
String str=”just for test”;
char char_arr[]=str.toCharArray();

或者
用String的charAt(index)方法可以获得index处的字符。
char a=str.charAt(index);

char转String:

注意：任何类型都可以用toString()方法转换为String。（基本类型不可以，基本类型对应的包装类可以,基本类型与其对应的包装类之间可以自动实现拆箱与装箱）。
Character a=new Character(‘a’);
String str=a.toString();

或者：
char a=’a’;
String str=a+””;

String类与整型之间相互转换

String转int：

String str=”123”;
int a=Integer.parseInt(str);
或者
int a=Integer.valueOf(my_str).intValue();

String 转Integer:

Integer a=Integer.valueOf(str);

int转String：

int a=12;
Integer num=a;
String str=num.toString();
或者：
int a=12;
String str=a+””;
或者：
int a=12;
String str=String.valueOf(a);
