leetcode-283-movezeros

  • LeetCode

题目

Given an array nums, write a function to move all 0’s to the end of it while maintaining the relative order of the non-zero elements.

For example, given nums = [0, 1, 0, 3, 12], after calling your function, nums should be [1, 3, 12, 0, 0].

Note:
You must do this in-place without making a copy of the array.
Minimize the total number of operations.

解法1

解法2

分析

本题不难，分析后，我们要做的就是将一个数组中的零元素放到数组后面，而将非零元素放在数组前面，并不改变原数组中非零元素
的相对位置。

第一种方法，相对简单一点，就是按顺序遍历整个数组，碰到非零元素就将非零元素放到原数组中，非零元素先不管，被覆盖也无所
谓，并用标识符flag记录非零元素的数量，将所有数组元素遍历一遍后，便把所有的非零元素置于数组前面了，然后从数组中下标为flag的位开始为数组赋值 0，直到将数组填满。

第二种方法，代码量相对较多，单比较容易理解，思想就是遍历数组，将所有的非零元素放到另一个数组暂存，然后再将暂存非零元
素的数组的值逐一赋给原数组，再将原数组的剩余位用零填满。
