leetcode-237-Delete-Node

  • LeetCode

题目

Write a function to delete a node (except the tail) in a singly linked list, given only access to that node.

Supposed the linked list is 1 -> 2 -> 3 -> 4 and you are given the third node with value 3, the linked list should
become 1 -> 2 -> 4 after calling your function.

解

分析

首先把题意搞懂，自己刚开始做的没把题意读明白就写代码，然后就懵了，感觉不对，甚至以为题目有问题。今早写博客，重新看提
时，恍然大悟，是自己太菜了。
题意：删除单项链表中的一个节点，所给的是要删除的节点。
写一个可以删除单项链表中特定节点的方法，该方法的参数是要删除的节点。预先声明了 listnode类，该类有两个属性，val和
next。也就是说通过一个节点的值和它所连接的下一个节点便可确定该节点，因此要删除一个节点将这两个属性的值更改为其下个节
点的对应的值就可以了。
