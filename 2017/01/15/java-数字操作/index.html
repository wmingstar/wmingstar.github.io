Math类

Math类中的所有方法都不需要实例变量值，因为这些方法是静态的，所以无须Math的实例，用类本身就好。

以类的名称调用静态方法，以实例的名称调用非静态方法。

获取实例的方法

（1）new()
（2）序列化
（3）java reflection API

注意

（1）静态方法不能调用非静态变量；因为静态方法是通过类的名称来调用的，无须实例，因此无法引用任何实
例变量。

Math类中常用的一些方法

Math.random()

返回介于 0.0~1.0之间的双精度浮点数

Math.abs()

返回双精度浮点数类型参数的绝对值，这个方法有覆盖的版本，传入整型会返回整型，传入双精度浮点会返回
双精度浮点。

Math.round()

根据参数是浮点数或双精度浮点数返回四舍五入后的整型或长整型。

Math.min()

返回两参数中较小的那个，有int、long、float、double的覆盖版本。
