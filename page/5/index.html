程序分析

根据题目，分析可知，每个月兔子的数量规律为数列 1,1,2,3,5,8,13,21….
第1、2个月数量均为1，第3个月开始兔子的数量为前两个月的和，因此知此题用递归比较合适。

第4题

将一个正整数分解质因数。例如：输入90,打印出90=233*5。

程序分析

对n进行分解质因数，先找到一个最小的质数k，然后按以下步骤执行：
（1）如果n==k,说明分解质因数的过程已经结束，打印出即可。
（2）如果n>k，且n能被k整除，则应打印出k的值，并用n除以k的商作为新的正整数n重复执行第一步。
（3）如果n不能被k整除，则将k+1作为k的值执行第一步。

第5题

程序分析

最大公约数为与最小公倍数之间有一定的关系，两数的最小公倍数等于两数的积除以两数的最大公约数，
所以此题的关键是求两数的最小公倍数，用辗转相除法。

JAVA进阶——类型转换

在学习java过程中，有些知识很简单，但是却很重要。这些简单的知识是学好javaa的基石，需要认真对待。在做刷题过程中经常遇到一些需要进行类型转换的问题，所以对类型转换相关的知识进行整理如下。

String类与字符型相互转换

String转char:

用String的toCharArray()方法可以将字符串转换为字符数组。
String str=”just for test”;
char char_arr[]=str.toCharArray();

或者
用String的charAt(index)方法可以获得index处的字符。
char a=str.charAt(index);

char转String:

注意：任何类型都可以用toString()方法转换为String。（基本类型不可以，基本类型对应的包装类可以,基本类型与其对应的包装类之间可以自动实现拆箱与装箱）。
Character a=new Character(‘a’);
String str=a.toString();

或者：
char a=’a’;
String str=a+””;

String类与整型之间相互转换

String转int：

String str=”123”;
int a=Integer.parseInt(str);
或者
int a=Integer.valueOf(my_str).intValue();

String 转Integer:

Integer a=Integer.valueOf(str);

int转String：

int a=12;
Integer num=a;
String str=num.toString();
或者：
int a=12;
String str=a+””;
或者：
int a=12;
String str=String.valueOf(a);
