leetcode-409-Longest-Palindrome

题目

Given a string which consists of lowercase or uppercase letters, find the length of the longest palindromes that can be built with those letters.

This is case sensitive, for example “Aa” is not considered a palindrome here

解

分析

回文字符串：正读和倒着读都一样的字符串。
最初读此题有疑惑：比如，输入为”aaabbbcc”，’a’和’b’的个数均为奇数，以为此时最长为5，其实此时最长为7，可以余下一个’a’或者’b’不用。

扩展

java

一维数组的声明方式：

type var[]; 或type[] var;
声明数组时不能指定其长度（数组中元素的个数），
Java中使用关键字new创建数组对象，格式为：
数组名 = new 数组元素的类型 [数组元素的个数]

初始化：

1.动态初始化：数组定义与为数组分配空间和赋值的操作分开进行；

2.静态初始化：在定义数字的同时就为数组元素分配空间并赋值；

3.默认初始化：数组是引用类型，它的元素相当于类的成员变量，因此数组分配空间后，每个元素也被按照成员变量的规则
被隐士初始化。

leetcode-169-Majority-Element

题目

Given an array of size n, find the majority element. The majority element is the element that appears more than ⌊ n/2 ⌋ times.

You may assume that the array is non-empty and the majority element always exist in the array.

解

分析

1.分别统计数组中的元素个数，将个数大于length/2的元素返回
2.将数组排序，将排序后的下标为length/2的元素返回

扩展

python

1.用collections.Counter()统计各元素的个数，返回字典。
2.sort()是容器的函数，可将容器中的元素排序；sorted()为内置函数，可对可迭代类型排序。
3.nums.count(x)也可得到nums中某元素的个数。

java

1.array.length属性得到数组长度。
2.Arrays类的sort()方法可以对数组进行排序。

leetcode-242-Valid-Anagram

题目

Given two strings s and t, write a function to determine if t is an anagram of s.

For example,
s = “anagram”, t = “nagaram”, return true.
s = “rat”, t = “car”, return false.

解

分析

重要的是读题，明白题目什么意思。题目的意思，输入两个字符串，来比较两个字符串是否是 anagram 关系。
(anagram 关系：两个字符串由相同字符且字符数相同的字符构成，只是字符的顺序不同。)
思路：将两个字符串中的字符进行排序，然后比较排序后的字符串是否相同就可以。

扩展

sort()函数是容器对象的函数，在容器内排序，无返回值，字符串对象没有该函数。
sorted()函数是内置函数，可对可迭代类型进行排序，并返回排好序的容器。

Counter是collections模块中的计数器类，可以用来计数，输入是字典，其他 iterable 的数据类型，还有命名的参数对。
输出是字典，其中元素作为key，其计数作为value。
